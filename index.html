📤 Pousser vers Firebase

Envoie l'état actuel (membres, config, aides...) vers Firebase. Écrase les données distantes.

📥 Tirer depuis Firebase

Recharge toutes les données depuis Firebase. Écrase les données locales.

⏸ Mode pause — synchronisation Firebase
▶ ACTIVE

Suspend les listeners Firebase. Les données ne se mettent plus à jour en temps réel. Utile pour tester sans interférence.

💾 Télécharger une sauvegarde

Exporte toutes les données Firebase en fichier .json. À conserver précieusement.

📂 Charger une sauvegarde

Importe un fichier .json précédemment exporté. Écrase toutes les données actuelles.

🔍 Tester la connexion Firebase
ℹ️ Comment ça fonctionne
Temps réel — Firebase synchronise les données automatiquement entre tous les appareils connectés.
Sauvegarde manuelle — Téléchargez régulièrement un fichier JSON comme backup local.
Restauration — En cas de problème, rechargez un fichier JSON pour restaurer toutes vos données.
Mode pause — Suspend uniquement les listeners temps réel, la base reste intacte.
